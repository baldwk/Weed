# Roadmap

## Phase 1: 工程骨架

目标：建立可运行的 Weed 桌面应用。

交付：

- Solution 和项目结构。
- WPF 主程序。
- 日志。
- 配置加载。
- 托盘图标。
- 主窗口基础样式。
- `Alt+Space` 唤起。

验收：

- 应用可启动和退出。
- 主窗口可通过快捷键打开。
- 设置文件可读写。
- 日志能记录启动和关闭。

## Phase 2: 主搜索体验

目标：形成 Alfred 风格输入和结果交互。

交付：

- 搜索输入框。
- 结果列表。
- 键盘导航。
- 默认动作执行。
- 基础主题 token。
- 空状态和错误状态。

验收：

- 输入时结果区域稳定刷新。
- 上下键、回车、Esc 可用。
- 窗口打开和关闭体验流畅。

## Phase 3: 插件运行时

目标：加载 managed DLL 插件并执行查询和命令。

交付：

- `Weed.Abstractions`。
- manifest schema。
- 插件扫描。
- `AssemblyLoadContext` 加载。
- 插件生命周期。
- QueryProvider 和 CommandHandler 分发。
- 插件异常日志。

验收：

- 示例插件可被发现、加载、查询和执行命令。
- 禁用插件后不参与查询。
- 插件异常会被记录并展示在插件详情中。

## Phase 4: 快捷键和查询路由

目标：完成三种入口和统一排序。

交付：

- Keyword 路由。
- Hotkey 注册和配置。
- ImplicitQuery 路由。
- 三项排序公式。
- 用户历史记录。
- 插件优先级配置。

验收：

- `clip xxx` 能命中剪切板插件。
- `Shift+Ctrl+C` 能触发剪切板面板。
- `Shift+Alt+A` 能触发截图。
- 无前缀结果按匹配分、历史分和优先级排序。
- 用户修改插件优先级后排序变化可见。

## Phase 5: AppLauncher

目标：无前缀搜索并打开开始菜单程序。

交付：

- 开始菜单索引。
- 快捷方式解析。
- 名称、拼音、缩写和模糊匹配。
- 图标加载。
- 打开应用动作。
- 打开所在位置和复制路径动作。

验收：

- 输入部分程序名可以找到应用。
- 输入拼音或首字母可以找到中文应用。
- 回车可以打开选中应用。
- 常用应用被选择后排序上升。

## Phase 6: Calculator

目标：无前缀计算表达式。

交付：

- 表达式识别。
- 表达式求值。
- 中文和全角符号归一化。
- 结果复制。
- 粘贴结果动作。

验收：

- 输入 `1+2*3` 显示 `7`。
- 输入 `sqrt(9)` 显示 `3`。
- 回车复制结果。
- 计算结果参与 ImplicitQuery 排序。

## Phase 7: Clipboard

目标：完成剪切板历史插件。

交付：

- Resident 插件生命周期。
- Windows 剪切板变化监听。
- 文本、图片、文件、RTF/HTML 摘要采集。
- SQLite 和 FTS 存储。
- `clip` 查询。
- `Shift+Ctrl+C` 历史面板。
- 复制、粘贴、删除、置顶。
- 保留策略和清理。

验收：

- 复制文本后可在 Weed 中搜索到。
- 复制图片后可预览和再次复制。
- 文件列表可记录和恢复。
- 历史面板可搜索、删除和置顶。

## Phase 8: Screenshot

目标：完成带 UI 的截图插件。

交付：

- 区域截图覆盖层。
- 多显示器支持。
- 截图编辑器。
- 画笔标注。
- 矩形框和圆形标注。
- 颜色和线宽选择。
- 复制和保存。
- 滚动截图流程。
- 滚动截图拼接预览。

验收：

- `Shift+Alt+A` 可进入区域截图。
- 截图后可添加画笔、矩形和圆形标注。
- 标注后可复制到剪切板。
- 标注后可保存到文件。
- 滚动截图可生成长图并进入编辑器。

## Phase 9: 开发者体验

目标：让第三方开发者可以编写插件。

交付：

- 插件模板。
- manifest schema 文档。
- API 文档。
- 示例插件。
- 打包说明。
- 调试说明。
- 插件日志查看。

验收：

- 新插件可通过模板创建。
- 插件能声明 Keyword、Hotkey 和 ImplicitQuery。
- 插件能返回结果并执行命令。
- 文档覆盖从创建到安装的完整流程。

## Phase 10: 发布打磨

目标：形成可公开试用的 MVP。

交付：

- 安装包。
- 开机启动设置。
- 自动更新方案。
- 崩溃日志。
- 性能基准。
- 主题打磨。
- 插件设置页完善。

验收：

- 全量安装和卸载流程可用。
- Windows 10 上完成核心工作流。
- 日常操作稳定。
- 首批用户可以通过文档开发插件。
